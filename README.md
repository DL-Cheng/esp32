# esp32

這是 ESP32 專案範例，同時也展示了使用 GitHub Codespaces 建立與更新靜態網頁的流程。

## 網頁內容
- `index.html`：專案首頁，介紹 ESP32 特性與相關連結。
- `word-learning.html`：單字學習練習頁面，支援上傳 `word.csv`，A 欄為中文、B 欄為英文。

## 單字學習功能
1. 使用者上傳 `word.csv` 檔案。
2. 顯示中文題目，使用者輸入英文答案並驗證是否正確。
3. 答對進入下一題；答錯則顯示正確答案並將該單字加入下一輪題目中增加出現次數。
4. 每個單字至少出現一次，直到所有題目全部做過。
5. 提供「重置遊戲」按鈕，可重新開始練習。

> 建議將 `word.csv` 以 UTF-8 編碼儲存，避免中文顯示亂碼。
## 連結
- 首頁：`index.html`
- 單字學習：`word-learning.html`
